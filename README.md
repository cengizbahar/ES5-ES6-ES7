## var , let , const


- Let : Değiştirilebilir
- const : Değiştirilemez
```
var x;
x = "mustafa";
console.log(x);

let fullname="Cengo";
console.log(fullname);
fullname="bahar";
console.log(fullname);

const email="cengiz@gmail.com";
console.log(email);
```
